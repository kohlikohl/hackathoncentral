Clouds Over London

Our Idea

To build that an app that provides segmentations of society with a breakdown on the suitability of areas in London to live.

Use data served up by the esd-toolkit to make an API mashup that we leverage in a dashboard.

Our tech

Front-end: Google Closure, Google Maps API Back-end: Node.js, Redis, Express.js, Mustache Hosting: Heroku
